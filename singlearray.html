for(j=i+1;j<5;j++){
if(array[i]>array[j]){
temp=array[j];
array[j]=array[i];
array[i]=temp;}
}
}

for(i=0;i<5;i++){
printf("%d ",array[i]);
}
}

Enter the array(size limit: 5):

2.)Value: 4
3.)Value: 3
4.)Value: 5
5.)Value: 2
Printing the array in ascending order:
1 2 3 4 5
